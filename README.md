## 📌 About This Project

This project is a **fully interactive HR Analytics Dashboard** built in Microsoft Excel. It was designed to help HR teams at Sunrise Technologies track workforce data, monitor performance, and make data-driven decisions — all from a single dashboard.

The dataset contains **538 employee records** with 13 attributes per employee, connected to a dynamic dashboard using Excel formulas, Pivot Tables, and Charts.

---

## 🗃️ Dataset Overview

| Column | Description |
|---|---|
| Emp ID | Unique employee identifier (EMP-2001 to EMP-2538) |
| Employee Name | Full name of employee |
| Department | Marketing, HR, Sales, IT, Finance, Operations, Admin, Customer Support |
| Branch | Mumbai, Pune, Thane, Nagpur, Nashik, Aurangabad, Kolhapur, Navi Mumbai |
| Designation | Job title / role |
| Salary (₹) | Monthly salary in Indian Rupees |
| Performance Rating | Excellent / Good / Average / Below Average |
| Joining Month | Month of joining (Jan–Dec) |
| Years of Experience | Total work experience |
| Gender | Male / Female |
| Employment Type | Permanent / Contractual / Intern |
| Leave Taken | Number of leaves taken |

---

## 📊 Dashboard Features

- ✅ **Department-wise Employee Count** — See headcount across all 8 departments
- ✅ **Performance Rating Distribution** — Breakdown of Excellent / Good / Average / Below Average
- ✅ **Branch-wise Salary Analysis** — Compare average salaries across 8 city branches
- ✅ **Gender Distribution** — Male vs Female ratio across the organization
- ✅ **Employment Type Split** — Permanent vs Contractual vs Intern
- ✅ **Leave Trends** — Analyze leave patterns by department or branch
- ✅ **Interactive Filters** — Slicers to filter by Department, Branch, Gender, and Rating

## 🚀 How to Use

1. Download `project_HR_Dashboard.xlsx`
2. Open in **Microsoft Excel** (2016 or later recommended)
3. Go to the **Dashboard** sheet
4. Use the **slicers** on the left/top to filter by Department, Branch, Gender, or Rating
5. Charts and KPIs update automatically based on your selection
